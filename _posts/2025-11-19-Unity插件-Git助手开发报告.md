## 使用说明

### Fire Git Assistant

- 通过 `Tools/Fire/Git Assistant` 打开工具窗口，左侧树状视图自动依据 `git status --porcelain` 结果按目录归类所有改动。
- 顶部信息栏实时显示当前分支、远端地址与最近提交摘要，便于核对工作状态。
- 操作区提供常用 Git 按钮：
  - **Refresh**：刷新状态、分支与日志。
  - **Stage All**：执行 `git add -A`，一次性暂存全部改动。
  - **Commit**：输入中文提交信息后提交。
  - **Push**：推送到远端，可指定 remote 与 branch。
  - **Pull**：从远端同步最新代码。
- 底部日志窗以 `git log --graph --oneline` 呈现分支结构，方便快速回顾历史。

### 安装与打包

1. 将 `Packages/com.fire.gitassistant` 保存在项目中，打开 Package Manager 便可通过 UPM 引用。
2. 若需分发或售卖：
   - **UPM Git Tag**：把该目录推送至独立仓库并创建 release。
   - **unitypackage**：在 Unity 中选中 `Packages/com.fire.gitassistant`，执行 `Assets > Export Package...` 导出。

### 使用建议

- 确保系统已安装 `git` 且命令行可调用。
- Unity 编辑器需具备项目根目录的读写权限。
- 窗口右上角齿轮按钮可切换语言，并配置默认远端与分支。

### 自定义扩展

- UI 逻辑集中在 `Editor/GitAssistantWindow.cs`，可自行扩展界面元素。
- Git 命令封装位于 `Editor/GitProcessUtility.cs`，可添加 `git tag`、`git stash` 等操作。
- 树视图实现见 `Editor/GitTreeView.cs`，支持扩展右键菜单、细粒度暂存等功能。

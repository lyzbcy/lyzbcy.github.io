# 一键部署 OpenClaw 使用指南

## 🚀 快速开始

### 1. 双击启动
双击 `🚀 启动小龙虾控制中心.bat`，浏览器会自动打开控制台。

### 2. 配置 API Key
在控制台中选择你常用的 AI 平台，填写 API Key：
- ✅ Google Gemini（推荐·大额免费）
- ✅ DeepSeek 深度求索
- ✅ Moonshot / Kimi
- ✅ Qwen 通义千问
- ✅ OpenAI
- ✅ Anthropic (Claude)
- ✅ Groq
- ✅ Perplexity
- ✅ 讯飞星火

### 3. 一键启动
点击「一键启动」按钮，OpenClaw 网关自动运行！

### 4. 开始使用
- 打开 QQ，发送消息到你的机器人
- OpenClaw 会自动回复
- 支持语音转文字、图片识别、文件处理等功能

## 📖 详细使用指南

### 第一次使用流程
1. **下载安装包** - 从 GitHub 下载最新版一键部署包
2. **解压文件** - 将压缩包解压到任意目录（建议英文路径）
3. **双击启动** - 运行 `🚀 启动小龙虾控制中心.bat`
4. **配置密钥** - 在控制台中添加你的 AI 平台 API Key
5. **启动网关** - 点击「一键启动」按钮
6. **开始对话** - 在 QQ 中向机器人发送消息

### 控制台功能说明
- **API 管理** - 添加、编辑、删除 API Key
- **模型选择** - 为不同平台选择合适的模型
- **网关控制** - 启动、停止、重启网关服务
- **日志查看** - 实时查看网关运行日志
- **额度查询** - 查看各 API 的剩余额度

## ❓ 常见问题

### Q1：双击 `.bat` 文件后没有反应？
**A**：可能是以下原因：
1. **PowerShell 被禁用**：右键 `.bat` 文件 →「以管理员身份运行」
2. **Node.js 缺失**：检查 `node_bin\node.exe` 是否存在
3. **网络问题**：查看终端是否有「连接超时」提示

**解决方法**：
- 右键 `.bat` 文件 →「打开方式」→「记事本」→ 复制内容
- 打开 PowerShell，粘贴并运行
- 查看终端输出的错误信息

---

### Q2：浏览器没有自动打开？
**A**：手动访问 `http://127.0.0.1:18789/`（或终端显示的端口号）

---

### Q3：API Key 配置失败？
**A**：检查以下几点：
1. **API Key 是否正确**：复制粘贴，不要手动输入
2. **网络是否正常**：尝试访问对应平台的官网
3. **防火墙是否拦截**：暂时关闭防火墙重试

**支持的平台**：
- Google Gemini：https://aistudio.google.com/
- DeepSeek：https://platform.deepseek.com/
- Moonshot：https://platform.moonshot.cn/
- Qwen：https://dashscope.console.aliyun.com/
- OpenAI：https://platform.openai.com/
- Anthropic：https://console.anthropic.com/
- Groq：https://console.groq.com/
- Perplexity：https://www.perplexity.ai/
- 讯飞星火：https://console.xfyun.cn/

---

### Q4：配置成功后网关启动失败？
**A**：可能是端口冲突：
1. **检查 18789 端口是否被占用**：
   ```powershell
   netstat -ano | findstr :18789
   ```
2. **终止占用进程**：
   ```powershell
   taskkill /F /PID <进程ID>
   ```
3. **重启 `.bat` 文件**

---

### Q5：网关启动后无法回复消息？
**A**：检查以下几点：
1. **QQ 机器人是否在线**：查看 QQ 机器人状态
2. **消息是否被过滤**：检查是否发送了敏感词
3. **权限是否足够**：确保机器人有「群聊/私聊」权限

---

### Q6：如何查看网关日志？
**A**：终端会实时输出日志，包括：
- 网关启动状态
- 消息接收记录
- 错误信息

---

### Q7：如何停止网关？
**A**：三种方法：
1. **关闭浏览器窗口**（推荐）
2. **在 PowerShell 中按 `Ctrl+C`**
3. **任务管理器中终止 `node.exe` 进程**

---

### Q8：支持哪些功能？
**A**：OpenClaw 支持：
- ✅ 自动回复消息
- ✅ 语音转文字（需要配置 STT）
- ✅ 图片识别（需要配置 OCR）
- ✅ 文件处理（需要配置相关技能）
- ✅ 定时任务（需要配置 Cron）

---

### Q9：如何扩展功能？
**A**：OpenClaw 支持「技能」扩展：
1. 访问 [ClawHub](https://clawhub.com/) 下载技能
2. 在控制台中「安装技能」
3. 重启网关生效

---

### Q10：API Key 安全吗？
**A**：**绝对安全！**
- API Key 仅存储在本地
- 不会上传到任何服务器
- 不会分享给第三方

---

### Q11：如何备份配置？
**A**：复制以下文件：
```
C:\Users\<用户名>\.openclaw\
├── agents\
│   └── main\
│       └── agent\
│           └── auth-profiles.json  # API Key
└── openclaw.json                   # 配置
```

---

### Q12：如何修改默认模型？
**A**：在控制台中「应用配置」时选择模型，或手动编辑：
```
C:\Users\<用户名>\.openclaw\openclaw.json
```

---

### Q13：如何更新 OpenClaw？
**A**：重新运行 `🚀 启动小龙虾控制中心.bat`，会自动检测更新。

---

### Q14：如何卸载？
**A**：删除 `一键部署openclaw` 文件夹即可（所有数据都在本地）。

---

### Q15：支持哪些平台的机器人？
**A**：目前支持：
- ✅ QQ 机器人（私聊/群聊）
- ✅ 待扩展其他平台

---

### Q16：如何配置语音转文字？
**A**：需要额外配置 Whisper 模型：
1. 下载 Whisper 模型文件
2. 在控制台中配置 STT 相关参数
3. 重启网关生效

---

### Q17：如何配置图片识别？
**A**：需要额外配置 OCR 或图像模型：
1. 在控制台中启用图片识别功能
2. 配置相应的 API 或本地模型
3. 重启网关生效

---

### Q18：网关启动后端口冲突怎么办？
**A**：修改默认端口：
1. 打开 `hub_server.js`
2. 找到 `const port = 18789;`
3. 修改为其他端口号（如 `18790`）
4. 重启 `.bat` 文件

---

### Q19：如何设置开机自启动？
**A**：将 `.bat` 文件快捷方式放入：
```
C:\Users\<用户名>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```

---

### Q20：支持离线使用吗？
**A**：部分功能支持离线：
- ✅ 本地模型（如 Whisper、本地 LLM）
- ❌ 云端 API 需要联网

## 常见吵架流程图

```
💔 因为一件小事，双方发生矛盾
           ↓
       🙏 一方道歉
           ↓
    ┌─────────────────┐
    │ 另一方是否接受?  │
    └─────────────────┘
           ↓
    ┌──────┴──────┐
    │             │
  接受          不接受
    │             │
    ↓             ↓
✅ 和解      ⚔️ 要把对错争明白
                ↓
          🔥 两个人争对错
                ↓
    💥 一方情绪爆发
    说狠话/用疏离作为制裁
                ↓
         ┌────────────────┐
         │ 另一方如何应对? │
         └────────────────┘
                ↓
         ┌──────┴──────┐
         │             │
    冷静处理      要扳赢对局
         │             │
         ↓             ↓
    🛡️ 避免升级   ⚡ 用更大力度的制裁
                     说狠话/关系威胁
         │             ↓
         │      ⚠️ 到达关系临界点
         │             ↓
         │      ┌──────────────┐
         │      │ 是否及时止损? │
         │      └──────────────┘
         │             ↓
         │      ┌──────┴──────┐
         │      │             │
         │      是            否
         │      │             │
         │      ↓             ↓
         │  💔 关系修复    💣 关系破裂
         │
         └──────────→ ✅ 和解
```

        <h1>⚔️ C++ 刷题特种兵突击指南</h1>

        <div class="alert-box">
            <strong>核心思想：</strong> 我们不学 C++ 开发，我们只是给 C 语言穿上一层 <strong>STL (标准库)</strong> 的外骨骼。除了以下 4
            个板块，其他（面向对象、继承、多态、文件流）统统不看！
        </div>

        <!-- 模块 1 -->
        <div class="module">
            <div class="module-header">
                <span>🎯 板块一：彻底告别指针噩梦 (引用)</span>
                <span class="time-tag">耗时：30分钟</span>
            </div>
            <div class="module-content">
                <p>在 C 语言刷题时，`int* returnSize` 这种写法很烦人。C++ 引入了 **引用 (&)**，让你像操作普通变量一样操作指针。</p>
                <ul>
                    <li><strong>必看视频：</strong>P89 - P91 (引用的基本语法 / 引用做函数参数)</li>
                    <li><strong>学会这招即可：</strong></li>
                </ul>
                <div class="code-snippet">
<span class="keyword">void</span> <span class="func">swap</span>(<span class="keyword">int</span> &a, <span class="keyword">int</span> &b) { <span class="type">// 加上 & 就行了，不用写 *a</span>
    <span class="keyword">int</span> temp = a;
    a = b;
    b = temp;
}
                </div>
            </div>
        </div>

        <!-- 模块 2 -->
        <div class="module">
            <div class="module-header">
                <span>📦 板块二：数组升级 (Vector)</span>
                <span class="time-tag">耗时：45分钟</span>
            </div>
            <div class="module-content">
                <p>刷题 80% 的情况都在用数组。C++ 的 <code>vector</code> 是会自动扩容的数组。</p>
                <ul>
                    <li><strong>必看视频：</strong>P186 (vector基本概念) ➜ P188 (构造) ➜ P191 (插入和删除)</li>
                    <li><strong>核心API：</strong> <code>push_back()</code> (尾插), <code>pop_back()</code> (尾删),
                        <code>size()</code> (长度), <code>resize()</code></li>
                    <li><strong>学会这招即可：</strong></li>
                </ul>
                <div class="code-snippet">
vector&lt;<span class="keyword">int</span>&gt; v;
v.push_back(10); <span class="type">// 自动扩容，不用 realloc</span>
<span class="keyword">int</span> len = v.size(); <span class="type">// 也不用传 returnSize 了</span>
                </div>
            </div>
        </div>

        <!-- 模块 3 -->
        <div class="module">
            <div class="module-header">
                <span>⚡ 板块三：查表神器 (Map/Set)</span>
                <span class="time-tag">耗时：40分钟</span>
            </div>
            <div class="module-content">
                <p>解决“两数之和”这类问题的物理外挂。C 语言手写哈希表要半小时，C++ 只要一行。</p>
                <ul>
                    <li><strong>必看视频：</strong>P205 (set容器) ➜ P210 (map容器)</li>
                    <li><strong>注意：</strong>视频里讲的是 <code>set/map</code> (红黑树，有序)，刷题更常用 <code>unordered_set/map</code>
                        (纯哈希，无序，O(1))，用法 API 完全一样，名字改一下就行。</li>
                    <li><strong>核心API：</strong> <code>map[key] = val</code>, <code>find()</code>, <code>count()</code>
                    </li>
                </ul>
            </div>
        </div>

# 本地预览说明

## 启动方式

双击 `zeen-tools/一键启动前端.bat`，或在终端执行：

```bash
npm run preview:local
```

## 停止方式

双击 `zeen-tools/一键关闭前端.bat`，或在终端按 `Ctrl+C`。

## 默认地址

- 导航页: http://127.0.0.1:8091/nav
- 首页:   http://127.0.0.1:8091/
- AR 体验: http://127.0.0.1:8091/ar/

## 推荐抽查顺序

1. **导航页** (`/nav`) — 所有页面入口汇总
2. **首页** (`/`) — Jekyll 构建后的完整博客首页（需先 `bundle exec jekyll build`）
3. **AR 体验页** (`/ar/`) — 4 个 AR 应用入口卡片
4. **捏脸 AR** (`/ar/face-deform.html`) — 需要摄像头权限
5. **手势生花** (`/ar/hand-flower.html`)
6. **手势魔法** (`/ar/gesture-game.html`)
7. **AI 健身教练** (`/ar/fitness-coach.html`)

## 这个预览服务解决了什么问题

Jekyll 项目通常需要 `bundle exec jekyll s` 才能本地预览，依赖 Ruby 环境。本预览服务用 Node.js 直接服务源文件（自动去除 YAML front matter 和 Liquid 标签），无需 Ruby 即可快速预览页面内容。

**优先级逻辑**：如果 `_site/` 目录存在（已 Jekyll 构建），优先返回构建后的完整 HTML；否则返回源文件（去除 front matter）。

## 端口说明

默认端口 `8091`，可通过环境变量修改：

```bash
LOCAL_PREVIEW_PORT=8092 node local-preview-server.js
```

## 注意事项

- AR 应用页面需要 HTTPS 或 localhost 才能访问摄像头（浏览器安全策略）
- 完整博客效果（含侧边栏、主题样式）需要 `bundle exec jekyll s`
- 本预览服务适合快速查看页面结构和 AR 应用是否正常加载

# 捞鱼的世界 — 3D 世界本地预览说明

## 启动方式

双击 `world/zeen-tools/一键启动前端.bat`，或在 `world/` 目录下执行：

```bash
npm run dev
```

## 停止方式

双击 `world/zeen-tools/一键关闭前端.bat`，或在终端按 `Ctrl+C`。

## 默认地址

- 3D 世界首页: http://127.0.0.1:5174/world/
- Vite 资源入口: http://127.0.0.1:5174/world/index.html

## 推荐抽查顺序

1. **3D 世界首页** (`/world/`) — 进入可自由探索的 3D 场景
2. **标志性建筑（显示屏塔）** — 位于场景北端的圆柱形屏幕环塔，展示各专栏文章
3. **NPC 对话** — 点击专栏 NPC 查看相关文档和对话

## 这个预览服务解决了什么问题

Vite 开发服务器提供 HMR（热模块替换）支持，修改 `world/src/` 下的代码后页面会自动刷新，适合开发调试。一键启动脚本自动处理 Node.js 检测、依赖安装、端口等待和浏览器打开。

## 端口说明

默认端口 `5174`，与 `vite.config.js` 中的 `server.port` 配置一致。如需修改，请同时更新 `vite.config.js` 和 `zeen-tools/一键启动前端.bat` 中的 `DEV_PORT`。

## 技术栈

- **Three.js** — 3D 渲染引擎
- **Vite** — 开发服务器和构建工具
- **simplex-noise** — 地形生成噪声算法

## 注意事项

- 首次启动如果 `node_modules` 不存在，脚本会自动执行 `npm install`
- 3D 场景需要 WebGL 支持，建议使用现代浏览器（Chrome / Edge / Firefox）
- Vite 开发服务器模式下 `/` 根路径会重定向到 3D 世界首页（由 `site-dev/index.html` 实现）
